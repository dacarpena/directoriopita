# Logos de Empresas PITA

## Instrucciones

Añade aquí los logos de las empresas. Los logos se escalarán automáticamente.

### Formato de nombre
El nombre del archivo debe coincidir exactamente con el identificador de la empresa (sin espacios, en minúsculas):

| Empresa | Nombre del archivo |
|---------|-------------------|
| Biorizon Biotech | `biorizon-biotech.png` ✓ |
| Cajamar | `cajamar.png` ✓ |
| Caparrós Nature | `caparros-nature.png` ✓ |
| Dicsa | `dicsa.png` ✓ |
| Edificio Grupo Cajamar | `edificio-grupo-cajamar.jpg` ✓ |
| Hintes | `hintes.png` ✓ |
| Iberveg | `iberveg.png` ✓ |
| IDM | `idm.jpg` ✓ |
| L&D | `l-d.png` ✓ |
| Lab | `lab.jpg` ✓ |
| Mecánica 4h | `mecanica-4h.png` ✓ |
| Tecnova | `tecnova.png` ✓ |
| Única Group | `unica-group.png` ✓ |

### Especificaciones recomendadas
- **Formato**: PNG (preferido) o SVG
- **Tamaño**: Mínimo 64x64px, recomendado 128x128px
- **Fondo**: Transparente (preferido) o blanco
- **Proporción**: Cuadrado (1:1)

El sistema escalará automáticamente cualquier imagen a 24x24px manteniendo la proporción.
